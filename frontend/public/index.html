# Advanced Hotel Booking System A full-stack hotel booking application. ## Features * User authentication (registration/login). * Hotel browsing and searching. * Detailed hotel view. * Booking functionality (placeholder). * User profile management. ## Technologies * Frontend: React, React Router, Context API, react-datepicker, Axios * Backend: Node.js, Express, MongoDB, Mongoose, JWT, bcryptjs ## Setup 1. Clone the repository. 2. `cd backend && npm install` 3. Set up MongoDB. Create a `.env` file in `backend` and add `MONGO_URI` and `JWT_SECRET`. 4. `cd frontend && npm install` 5. Start backend: `cd backend && npm run dev` (or `node server.js`) 6. Start frontend: `cd frontend && npm start`
